# Shopify 集成规格

## 概述

与 Shopify API 集成，实现产品上传、多店铺管理和数据导出。

## 功能需求

### 1. 店铺管理

**店铺连接：**
- AI 询问用户当前要操作的店铺
- 用户提供 Shopify API Token
- 支持多店铺管理
- Token 本地缓存存储

**Token 存储：**
- 加密存储
- 店铺别名管理
- 支持添加/删除/更新店铺

### 2. 产品上传

**上传流程：**
1. 产品数据处理完成
2. 上传到 Shopify（草稿状态，非上架）
3. 用户二次确认后上架

**上传控制：**
- 使用 Shopify API
- 参考：https://shopify.dev/docs/apps/build/ai-toolkit
- 支持产品变体上传
- 支持多图上传

### 3. 产品匹配与更新

**匹配逻辑：**
- 通过 SKU 判断产品是否已存在
- 已存在产品跳过处理
- 记录跳过日志

**更新策略：**
- 跳过现有产品（不覆盖）
- 记录操作结果

### 4. 数据导出

**导出格式：**
- Excel 格式
- 包含所有产品字段
- 支持筛选导出

### 5. 本地缓存

**缓存内容：**
- 已处理产品信息
- 导入历史记录
- 操作日志

**数据库：**
- SQLite 本地数据库
- 支持查询历史记录

## 验收标准

- [ ] 成功连接 Shopify API
- [ ] 产品上传为草稿状态
- [ ] 多店铺切换正常
- [ ] Token 安全存储
- [ ] SKU 匹配逻辑正确
- [ ] 导出 Excel 格式正确

## 边界情况

- API Token 失效时的处理
- 网络超时时的重试
- 上传失败时的错误记录
- 店铺配额限制处理
