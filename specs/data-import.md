# 数据导入规格

## 概述

从多种数据源导入产品信息，支持 Excel 文件和网站链接。

## 功能需求

### 1. Excel 导入

- 支持标准 Excel 格式（.xlsx, .xls, .csv）
- 固定模板格式，但需要列映射配置
- 用户可指定哪列对应哪个字段

**必需字段映射：**
- 产品标题
- 产品描述
- 价格
- 库存
- SKU
- 图片 URL（多图用分隔符）
- 变体信息（颜色、尺寸等）
- 分类/标签

### 2. 网站链接导入

**支持的平台：**
- Amazon
- AliExpress
- 1688
- 独立站

**技术要求：**
- 模拟人为浏览获取产品信息
- 处理反爬机制
- 提取产品标题、描述、图片、变体、价格

### 3. 导入控制

- 每次导入上限：100 个商品
- 显示导入进度
- 失败重试机制（最多 3 次）
- 单个产品失败时继续处理其他产品

## 验收标准

- [ ] Excel 文件成功解析并映射字段
- [ ] 至少支持 2 个主流电商平台的产品抓取
- [ ] 导入进度实时显示
- [ ] 失败产品可重试

## 边界情况

- Excel 文件格式错误时的提示
- 网站无法访问时的处理
- 图片链接失效时的处理
- 数据不完整时的处理
