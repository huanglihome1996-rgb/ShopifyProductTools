# Web 界面规格

## 概述

提供 Web 界面供用户操作产品导入、优化和上传流程。

## 功能需求

### 1. 店铺管理页面

- 店铺列表展示
- 添加新店铺（输入店铺名、API Token）
- 编辑/删除店铺
- 当前店铺切换

### 2. 产品导入页面

**Excel 导入：**
- 文件上传
- 列映射配置界面
- 预览导入数据

**网站链接导入：**
- 批量输入链接
- 选择平台类型（Amazon/AliExpress/1688/独立站）
- 抓取进度显示

### 3. 产品优化页面

- 产品列表展示
- 单个/批量优化
- 优化前后对比
- 手动编辑优化结果

### 4. 产品上传页面

- 待上传产品列表
- 批量上传（最多 100 个）
- 上传进度显示
- 失败产品重试

### 5. 历史记录页面

- 导入历史
- 操作日志
- 错误报告
- 导出历史数据

### 6. 仪表盘

- 统计概览（已处理产品数、成功率等）
- 最近操作记录
- 待处理任务

## 技术要求

- 响应式设计
- 实时进度更新（WebSocket 或轮询）
- 错误提示友好
- 操作确认弹窗

## 验收标准

- [ ] 所有页面正常访问
- [ ] 文件上传功能正常
- [ ] 进度实时更新
- [ ] 错误信息清晰展示

## 边界情况

- 大文件上传超时处理
- 并发操作处理
- 会话超时处理
